
Git начало работы:
  Установка и настройка:
   1. После установки git на компьютер.
   2. Регострация:
     git config --global user.name "Andrey Galena"
     git config --global user.email andreigal5@gmail.com
     git config --list - проверка записи(выход Q).
  
  Начало работы:
   1. Создаём папку с проэктом.
   2. Заходим в неё через терминал.
   3. Инициализируем её с помощю команды:
     $git init - создаёт системную папку git в папке проэкта.
   4. При изменении проэкта в VsCode загарается ветка git(c лева).

   git status  - для проверки состаяния проэкта команда.

   git add name_file - следим за этим файлом(подгатавливаем).
   git add . - следим за всеми файлами в проэкте (подгатавливаем).
   git rm --cached name_file - прекратить следить.

   git commit -m "Коментарий"  - сохроняем (делаем слепок).

   5. Для списка игноруруемых файлов создаём папку:
     .gitignor - в дереве проэкта.
     В этом файле записуем:
      errors.txt  - запись по игнорированию файла.
      /logs       - запись по игнорированию папки.

  Ветки:
  
   git branch  - показывает в какой ветке сейчас находишся(master),
                 в VsCode показывает в низу с лева.
  
   git branch test    - создаём дополнительную ветку(test).
   git branch -D test - удаляем не нужную ветку.
  
   git checkout name  - переход с ветки на ветку.
   get checkout -b name - быстрое создание и переход на новую ветку.

   git merge name  - обьединяем две ветки в одну с ветки *master.

   4. Связываем Git с GitHub:( РАБОТАТЬ ТОЛЬКО НА АНГЛИСКОМ ПЕРЕВОДЕ )

   6. Обновляем страничку.

  Последовательность работы:
   1. Вносим изменения в проэкт.
   2. Обрабатываем: git add name_file  или git add .
   3. Делаем слепок проэкта: git commit -m "коментарий"
   4. Загружаем на GitHub: git push
   5. Проверяем изменился ли файл на GitHub.

  Работа с другого компьютера:
   1. Зайти на страничку проэкта в GitHub:
       1. Скачать код в zip формате.
       2. В папке где должен лижать проэкт:
            git clone https://github.com/AndreyGalena/git-course.git
       
       3. Заходим в папку с проэктом: cd имя_папки
       4. Изменяем проэкт в VsCode.
       5. Проверяем: git status
       6. git add .
       7. git commit -m "коментарий"
       8. git push
